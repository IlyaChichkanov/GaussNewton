# GaussNewton: идентификация параметров нелинейных динамических систем

Данный проект реализует **метод Гаусса–Ньютона** с **multiple shooting** для идентификации параметров и начальных условий нелинейных динамических систем по зашумлённым измерениям.  
Поддерживаются два режима вычислений: **NumPy/SciPy** (для отладки) и **JAX** (для ускорения через JIT-компиляцию).

## Возможности

- Символьное построение модели с помощью **CasADi**.
- Автоматическое вычисление якобианов правой части и функции измерений.
- Интегрирование расширенной системы (состояние + матрицы чувствительности).
- **Multiple shooting** с условиями непрерывности траектории.
- Регуляризация (L2) и метод расширенного Лагранжиана для устойчивого решения системы ККТ.
- Генератор синтетических данных для тестирования.
- Визуализация результатов: фазовые траектории, временные ряды всех состояний, сходимость параметров, невязки.
